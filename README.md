# Java_2D_Game (PD2-project)

## Environment

- Java
- Eclipse
- desktop platform

## Gameplay Instructions

1. Click to start the game.
2. Use AD keys to control movement direction.
3. Press the Space key to jump.
4. Press K key to attck.
